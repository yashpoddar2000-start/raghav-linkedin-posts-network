1. **"What raw data supports this topic?"**
   - For "Chipotle franchise": owned model metrics + franchise benchmarks
   - For "Taco Bell vs Pizza Hut": unit economics for both chains

2. **"What makes comparison valid?"**
   - Same metrics for both sides (revenue/store for both, not revenue for one and profit for other)
   - Same time periods (both 2024, not one 2024 and one 2020)
   - Same units ($, %, sq ft - not mixing currencies or measurements)

3. **"What validates the data?"**
   - Recent (Q3 2024, not 2019)
   - Official sources (10-K, not blog speculation)
   - Specific ($1.9B, not "around $2B")

4. **"Is the dataset complete for analysis?"**
   - Both sides represented (owned AND franchise models)
   - All key metrics present (revenue, profit, margins, etc.)
   - Context included (time periods, events like bankruptcies)
